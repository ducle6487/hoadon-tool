Công Cụ Hóa Đơn

Tự động tra cứu hóa đơn điện tử hàng loạt nhanh chóng

1. Hướng dẫn sử dụng

Tool đọc dữ liệu từ file Excel. Bạn bắt buộc cần có dòng đầu tiên liệt kê chính xác các Mã Số Cột sau:

  • nbmst: Mã số thuế người bán (Bắt buộc)
  • lhdon: Loại hóa đơn (Mặc định Hóa đơn GTGT)
  • khhdon: Ký hiệu hóa đơn (Bắt buộc)
  • shdon: Số hóa đơn (Bắt buộc)
  • tgtttbso: Tổng tiền thanh toán (Bắt buộc)
Tải file mẫu (Excel)
